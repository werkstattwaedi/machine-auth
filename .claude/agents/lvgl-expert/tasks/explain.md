# Explain Task

Provide an in-depth explanation of an LVGL widget or concept with practical examples.

## Process

### 1. Fetch Documentation

1. Read widget documentation at `third_party/lvgl/docs/widgets/`
2. Check widget header at `third_party/lvgl/src/widgets/lv_<widget>.h`
3. Look for examples in `third_party/lvgl/examples/`

### 2. Analyze Widget/Concept

Understand deeply:
- What problem does this widget solve?
- What are the key functions and properties?
- How does it integrate with styles and events?
- What are common usage patterns?
- What mistakes do people commonly make?

## Output Format

### Overview
What this widget/concept does and when to use it (2-3 paragraphs)

### Key APIs
Core functions and their purposes

### Basic Usage
Simple example showing the most common use case

```c
// Example code
```

### Styling
How to style this widget with `lv_style_t`

### Events
Common events to handle for this widget

### Advanced Patterns
More sophisticated usage examples

### Common Mistakes
What to avoid and why

### Related Widgets
Other widgets that work well with this one
